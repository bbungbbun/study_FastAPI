# 설치과정
- [Python 공식 사이트 방문](https://www.python.org/downloads/)
- 본인에게 맞는 버전 다운로드
- exe 파일 실행
- 설치 진행
- cmd 터미널 열어서 `python`입력

## 맥의 경우
- 별다른 설정 없이 설치 진행


## 윈도우의 경우
- `Add Python 3.9 to PATH` 체크박스 클릭 후 설치 진행


## cmd 터미널


### cmd 여는 법
- 단축키 : 윈도우키 + r
- 단축키를 눌렀을 때 뜨는 창에 `cmd` 입력 후 확인
![실행창](image.png)


### 파이썬이 잘 설치되었는지 확인하는 방법
- cmd창에 `python`이라고 입력
- 아래와 같이 파이썬 버전에 대한 정보가 나오면 다운로드 성공
```
Python 3.12.2 (tags/v3.12.2:6abddd9, Feb  6 2024, 21:26:36) [MSC v.1937 64 bit (AMD64)] on win32
Type "help", "copyright", "credits" or "license" for more information.

```

- 꺾쇠 표시 옆에 명령어를 입력하면 python 프로그램을 실행함
```
>>> print("hello")
hello
```
